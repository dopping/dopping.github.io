vscode插件整理

发表于 2018-05-28   |   更新于 2018-05-28 | 分类于 前端编译工具vscode

通用插件

HTML Snippets

超级实用且初级的 H5代码片段以及提示

HTML CSS Support

让 html 标签上写class 智能提示当前项目所支持的样式
新版已经支持scss文件检索

jQuery Code Snippets

jquery 重度患者必须品，废话不多说，上图

Path Intellisense

自动路劲补全，默认不带这个功能的，赶紧装

Document this

js 的注释模板 （注意：新版的vscode已经原生支持,在function上输入/** tab）

HTMLHint

html代码检测

beautify

格式化代码的工具

Atuo Rename Tag

修改 html 标签，自动帮你完成尾部闭合标签的同步修改，不过有些bug。

GitLens

丰富的git日志插件

fileheader

顶部注释模板，可定义作者、时间等信息，并会自动更新最后修改时间

ctrl + shift + i启用

Bracket Pair Colorizer

让括号拥有独立的颜色，易于区分。可以配合任意主题使用。

vue插件

vetur

语法高亮、智能感知、Emmet等

VueHelper

snippet代码片段

Import Cost

引入包大小计算,对于项目打包后体积掌握很有帮助

vscode主题篇

One Dark Pro

这是为 VS Code准备的 One Dark 主题，他是 Atom 标志性的主题。需要我说更多吗？我喜欢这个主题。

Monokai Pro

来自原 Monokai 的创作者，Monokai Pro是一套极具现代化的功能配色方案。让我们摆脱干扰，直接进入代码。

One Monokai Theme

多彩的主题！ 很多颜色，无处不在。

Panda Theme

vscode-icons (3,190,596 安装量)

Material Icon Theme (819,174 安装量)
