javascript常用的原生数组方法

合并数组

方法：concat()

描述：连接2个灬更多的数组 并放回结果

语法：arrayObject.concat(arrayX,arrayX,……,arrayX)

创建新数组(切割数组 返回新数组)

方法：slice()

描述：从已有的数组中返回选定的元素。

语法：array.slice(start, end)

连接数组

方法：join()

描述：将所有的数组元素连接成一个字符串。元素是通过指定的分隔符进行分隔的。

语法：arrayObject.join(separator)

参数：可选。指定要使用的分隔符。如果省略该参数，则使用逗号作为分隔符。

删除数组元素(最后一个元素) 并返回 改变元素组

方法：pop()

描述：删除并返回数组的最后一个元素。

语法：arrayObject.pop()

删除数组第一个元素

方法：shift()

描述：删除并返回数组的第一个元素。

语法：arrayObject.shift()

添加数组元素 (元素末尾添加)

方法：push()

描述：可向数组的末尾添加一个或多个元素，并返回新的长度。

语法：arrayObject.push(arr1,arr2,….,arrx)

在数组开头添加元素

方法：unshift()

描述：可向数组的开头添加一个或多个元素，并返回新的长度。

语法：arrayObject.push(arr1,arr2,….,arrx)

插入数组元素

方法：splice()

描述：向数组中添加/删除项目，然后返回被删除的项目。

语法：arrayObject.splice(index,howmany,item1,…..,itemX)
